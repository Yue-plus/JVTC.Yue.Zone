# Ubuntu Server 安装后配置

# 安装中文语言包

```shell
sudo apt install language-pack-zh-hans manpages-zh -y
localectl set-locale LANG=zh_CN.UTF-8 LANGUAGE=zh_CN.UTF-8
```

重新登入系统后生效。
