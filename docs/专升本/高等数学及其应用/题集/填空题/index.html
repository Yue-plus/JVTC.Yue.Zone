  3. 极限的指数运算：通过变形将问题转化为已知极限的形式。

解题思路：

  1. 分析极限形式：

    当 x→0x \to 0x→0 时，表达式 (1+x)2x(1+x)^{\frac{2}{x}}(1+x)x2​ 可以写为：

    lim⁡x→0(1+x)2x=lim⁡x→0[(1+x)1x]2\lim_{x \to 0} (1+x)^{\frac{2}{x}} = \lim_{x \to 0} \left[(1+x)^{\frac{1}{x}}\right]^2x→0lim​(1+x)x2​=x→0lim​[(1+x)x1​]2

  2. 应用已知极限：

    已知 lim⁡x→0(1+x)1x=e\lim \limits_{x \to 0} (1+x)^{\frac{1}{x}} = ex→0lim​(1+x)x1​=e，因此：

    lim⁡x→0[(1+x)1x]2=e2\lim_{x \to 0} \left[(1+x)^{\frac{1}{x}}\right]^2 = e^2x→0lim​[(1+x)x1​]2=e2

      (1−h)−2h=[(1−h)−1h]2→e2(1-h)^{-\frac{2}{h}} = \left[(1-h)^{-\frac{1}{h}}\right]^2 \to e^2(1−h)−h2​=[(1−h)−h1​]2→e2

曲线 y=x3−3x2+1y = x^3 - 3x^2 + 1y=x3−3x2+1 的拐点是 ____ 。

答案解析

答案：(1,−1)(1, -1)(1,−1)

考点解析：

  1. 二阶导数判断凹凸性
  2. 拐点的定义及求解步骤

解题思路：

要确定曲线 y=x3−3x2+1y = x^3 - 3x^2 + 1y=x3−3x2+1 的拐点，需验证二阶导数的零点及其两侧凹凸性变化。

  1. 求一阶导数：y′=3x2−6xy' = 3x^2 - 6xy′=3x2−6x
  2. 求二阶导数：y′′=6x−6y'' = 6x - 6y′′=6x−6
  3. 解方程 y′′=0y'' = 0y′′=0：6x−6=0⇒x=16x - 6 = 0 \Rightarrow x = 16x−6=0⇒x=1
  4. 验证凹凸性变化：
    • 当 x<1x < 1x<1 时（如 x=0x=0x=0），y′′=−6<0y'' = -6 < 0y′′=−6<0，曲线凹向下。
    • 当 x>1x > 1x>1 时（如 x=2x=2x=2），y′′=6>0y'' = 6 > 0y′′=6>0，曲线凹向上。
  5. 计算对应 yyy 值：代入原函数，y=13−3(1)2+1=−1y = 1^3 - 3(1)^2 + 1 = -1y=13−3(1)2+1=−1

答案：(0,14)(0, \frac{1}{4})(0,41​)

考点解析：

  1. 变上限积分的导数：根据微积分基本定理，ddx∫axf(t)dt=f(x)\frac{d}{dx} \int_{a}^{x} f(t) dt = f(x)dxd​∫ax​f(t)dt=f(x)。
  2. 导数的符号与单调性：若导数为负，函数在该区间单调递减。
  3. 不等式求解：解 f′(x)<0f'(x) < 0f′(x)<0 的区间。

解题思路：

  1. 求导
    根据微积分基本定理，原函数的导数为被积函数在 xxx 处的值：

    f′(x)=2−1xf'(x) = 2 - \frac{1}{\sqrt{x}}f′(x)=2−x​1​

  2. 解不等式

    由于 x>0x > 0x>0，解集为 0<x<140 < x < \frac{1}{4}0<x<41​。

    • 当 x>14x > \frac{1}{4}x>41​ 时，f′(x)>0f'(x) > 0f′(x)>0，函数递增； 当 x<14x < \frac{1}{4}x<41​ 时，f′(x)<0f'(x) < 0f′(x)<0，函数递减。

考点解析：

  1. 全微分的定义 ： dz=∂z∂xdx+∂z∂ydy\mathrm{d} z = \frac{\partial z}{\partial x} \mathrm{d}x + \frac{\partial z}{\partial y} \mathrm{d}ydz=∂x∂z​dx+∂y∂z​dy。

  3. 偏导数的计算 ： 对多变量函数分别对每个变量求导。

解题思路：

  1. 求偏导数 ∂z∂x\frac{\partial z}{\partial x}∂x∂z​：

  2. 求偏导数 ∂z∂y\frac{\partial z}{\partial y}∂y∂z​：

    ∂z∂y=1x3+y3⋅3y2=3y2x3+y3\frac{\partial z}{\partial y} = \frac{1}{x^3 + y^3} \cdot 3y^2 = \frac{3y^2}{x^3 + y^3}∂y∂z​=x3+y31​⋅3y2=x3+y33y2​

    • 计算分子： ∂z∂x∣(1,1)=3⋅122=32,∂z∂y∣(1,1)=3⋅122=32\frac{\partial z}{\partial x}\bigg|_{(1,1)} = \frac{3 \cdot 1^2}{2} = \frac{3}{2}, \quad \frac{\partial z}{\partial y}\bigg|_{(1,1)} = \frac{3 \cdot 1^2}{2} = \frac{3}{2}∂x∂z​​(1,1)​=23⋅12​=23​,∂y∂z​​(1,1)​=23⋅12​=23​
