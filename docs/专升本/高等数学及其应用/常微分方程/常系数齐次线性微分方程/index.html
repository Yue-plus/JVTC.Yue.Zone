常系数齐次线性微分方程

　　先讨论二阶常系数齐次线性微分方程的解法，再把二阶方程的解法推广到 nnn 阶方程。

　　在二阶齐次线性微分方程

y′′+P(x)y′+Q(x)y=0(7-1)\tag{7-1} y'' + P(x) y' + Q(x) y = 0y′′+P(x)y′+Q(x)y=0(7-1)

其中 p,qp, qp,q 是常数，那么称 (7-2) 为 二阶常系数齐次线性微分方程 。
如果 p,qp, qp,q 不全为常数，称 (7-1) 为 二阶变系数齐次线性微分方程 。
