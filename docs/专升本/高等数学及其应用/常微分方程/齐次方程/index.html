齐次方程

齐次方程​

　　如果一阶微分方程可化成

dydx=φ(yx)(3-1)\tag{3-1} \frac{\mathrm{d} y}{\mathrm{d} x} = \varphi \left( \frac{y}{x} \right)dxdy​=φ(xy​)(3-1)

的形式，那么就称这方程为 齐次方程，例如

(xy−y2)dx−(x2−2xy)dy=0(xy - y^2) \mathrm{d} x - (x^2 - 2xy) \mathrm{d} y = 0(xy−y2)dx−(x2−2xy)dy=0

是齐次方程，因为它可化成

dydx=xy−y2x2−2xy\frac{\mathrm{d} y}{\mathrm{d} x} = \frac{xy - y^2}{x^2 - 2xy}dxdy​=x2−2xyxy−y2​

即

dydx=yx−(yx)21−2(yx)\frac{\mathrm{d} y}{\mathrm{d} x} = \frac{\frac{y}{x} - \left( \frac{y}{x} \right)^2}{1 - 2 (\frac{y}{x})}dxdy​=1−2(xy​)xy​−(xy​)2​

　　在齐次方程

dydx=φ(yx)\frac{\mathrm{d} y}{\mathrm{d} x} = \varphi \left( \frac{y}{x} \right)dxdy​=φ(xy​)

中，引进新的未知函数

u=yx(3-2)\tag{3-2} u = \frac{y}{x}u=xy​(3-2)

就可把它化为可分离变量的方程。 因为由 (3-2) 有

y=ux,dydx=u+xdudxy = ux, \quad \frac{\mathrm{d} y}{\mathrm{d} x} = u + x \frac{\mathrm{d} u}{\mathrm{d} x}y=ux,dxdy​=u+xdxdu​

代入方程 (3-1)，便得方程

u+xdudx=φ(u)u + x \frac{\mathrm{d} u}{\mathrm{d} x} = \varphi (u)u+xdxdu​=φ(u)

即

xdudx=φ(u)−ux \frac{\mathrm{d} u}{\mathrm{d} x} = \varphi (u) - uxdxdu​=φ(u)−u

分离变量，得

duφ(u)−u=dxx\frac{\mathrm{d} u}{\varphi (u) - u} = \frac{\mathrm{d} x}{x}φ(u)−udu​=xdx​

两端积分，得

∫duφ(u)−u=∫dxx\int \frac{\mathrm{d} u}{\varphi (u) - u} = \int \frac{\mathrm{d} x}{x}∫φ(u)−udu​=∫xdx​

求出积分后，再以 yx\frac{y}{x}xy​ 代替 uuu ，便得所给齐次方程的通解。
