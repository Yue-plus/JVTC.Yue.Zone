函数极限的定义​

在自变量的某个变化过程中，如果对应的函数值无限接近于某个确定的数，那么这个确定的数就叫做在这一变化过程中 函数的极限 。

lim⁡x→x0f(x)=A或f(x)→A(当 x→x0)\lim_{x \to x_0} f(x) = A \quad \text{或} \quad f(x) \to A \quad (\text{当 } x \to x_0)x→x0​lim​f(x)=A或f(x)→A(当 x→x0​)

函数的左右极限​

函数的左右极限是指函数在某点附近左右两侧的极限值。 具体来说：

  1. 右极限：函数 f(x)f(x)f(x) 在 x=ax = ax=a 点的右极限是指当 xxx 从右侧逼近 aaa 时，f(x)f(x)f(x) 的极限值，记作 lim⁡x→a+f(x)\lim \limits_{x \to a^+} f(x)x→a+lim​f(x)。 数学表达为：lim⁡x→a+f(x)=L\lim \limits_{x \to a^+} f(x) = Lx→a+lim​f(x)=L 其中， LLL 是函数在该点右侧的极限值。

  2. 左极限：函数 f(x)f(x)f(x) 在 x=ax = ax=a 点的左极限是指当 xxx 从左侧逼近 aaa 时，f(x)f(x)f(x) 的极限值，记作 lim⁡x→a−f(x)\lim \limits_{x \to a^-} f(x)x→a−lim​f(x)。 数学表达为：lim⁡x→a−f(x)=L\lim \limits_{x \to a^-} f(x) = Lx→a−lim​f(x)=L 其中， LLL 是函数在该点左侧的极限值。

函数极限存在的充分必要条件​

一个函数在某点的极限存在的充分必要条件是该点 左右极限都存在且相等 。 用数学表示为：

设函数 f(x)f(x)f(x) 在 x=ax = ax=a 处的极限为 LLL ，则满足以下条件：

  1. 右极限存在，即 lim⁡x→a+f(x)=L\lim \limits_{x \to a^+} f(x) = Lx→a+lim​f(x)=L
  2. 左极限存在，即 lim⁡x→a−f(x)=L\lim \limits_{x \to a^-} f(x) = Lx→a−lim​f(x)=L
  3. 右极限与左极限相等，即 lim⁡x→a+f(x)=lim⁡x→a−f(x)=L\lim \limits_{x \to a^+} f(x) = \lim \limits_{x \to a^-} f(x) = Lx→a+lim​f(x)=x→a−lim​f(x)=L

曲线渐近线​

曲线的渐近线是指当曲线趋近于无穷时，曲线逐渐靠近但不相交的直线。 渐近线有水平渐近线（横向渐近线）、垂直渐近线（纵向渐近线）和斜渐近线。

水平渐近线的求法​

水平渐近线通常存在于函数的图形中，并且其方程通常为 y=ky = ky=k，其中 kkk 是常数。 对于函数 y=f(x)y = f(x)y=f(x)，当 xxx 趋近于无穷大或无穷小时，如果 yyy 趋近于某个常数 kkk，则这个常数 kkk 所对应的直线 y=ky = ky=k 就是水平渐近线。 求水平渐近线的方法如下：

  1. 计算 lim⁡x→+∞f(x)\lim \limits_{x \to +\infty} f(x)x→+∞lim​f(x)，得到的极限值就是水平渐近线的 yyy 坐标；
  2. 计算 lim⁡x→−∞f(x)\lim \limits_{x \to -\infty} f(x)x→−∞lim​f(x)，得到的极限值也是水平渐近线的 yyy 坐标。

如果这两个极限存在且相等，则水平渐近线就是 y=ky = ky=k。

例如，对于函数 f(x)=1xf(x) = \frac{1}{x}f(x)=x1​，我们计算：

lim⁡x→+∞1x=0和lim⁡x→−∞1x=0\lim_{{x \to +\infty}} \frac{1}{x} = 0 \quad \text{和} \quad \lim_{{x \to -\infty}} \frac{1}{x} = 0x→+∞lim​x1​=0和x→−∞lim​x1​=0

因此，y=0y = 0y=0 是 f(x)=1xf(x) = \frac{1}{x}f(x)=x1​ 的水平渐近线。

垂直渐近线的求法​

垂直渐近线通常存在于函数的图形中，并且其方程通常为 x=hx = hx=h，其中 hhh 是常数。 对于函数 y=f(x)y = f(x)y=f(x)，如果在某个有限的 xxx 值 hhh 处，f(x)f(x)f(x) 的值趋近于无穷大或无穷小，则 x=hx = hx=h 就是垂直渐近线。 求垂直渐近线的方法如下：

  1. 找到函数的分母为零但分子不为零的点，因为这些点可能是函数的无穷点；
  2. 计算 lim⁡x→h+f(x)\lim \limits_{{x \to h^+}} f(x)x→h+lim​f(x) 或 lim⁡x→h−f(x)\lim \limits_{{x \to h^-}} f(x)x→h−lim​f(x)，如果极限值趋于无穷大或无穷小，则 x=hx = hx=h 就是垂直渐近线。

例如，对于函数 g(x)=1x−2g(x) = \frac{1}{x-2}g(x)=x−21​，我们找到函数的分母为零的点 x=2x = 2x=2，然后计算：

lim⁡x→2+1x−2=+∞和lim⁡x→2−1x−2=−∞\lim_{{x \to 2^+}} \frac{1}{x-2} = +\infty \quad \text{和} \quad \lim_{{x \to 2^-}} \frac{1}{x-2} = -\inftyx→2+lim​x−21​=+∞和x→2−lim​x−21​=−∞

因此，x=2x = 2x=2 是 g(x)=1x−2g(x) = \frac{1}{x-2}g(x)=x−21​ 的垂直渐近线。
