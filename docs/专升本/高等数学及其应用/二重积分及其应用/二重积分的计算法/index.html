按照二重积分的定义来计算二重积分，对少数特别简单的被积函数和积分区域来说是可行的， 但对一般的函数和区域来说，这不是一种切实可行的方法， 本节介绍一种计算二重积分的方法，这种方法是把二重积分化为两次定积分来计算。

交换二次积分的积分次序的方法​

1. 理解积分区域的几何意义​

二重积分的积分区域通常是某个平面区域 DDD ，你需要清楚该区域的边界形式。例如：

  • 直角区域（矩形）：通常可以直接交换积分次序。
  • 一般区域：需要分析区域的上下或左右边界，确定适合的积分次序。

2. 画出积分区域​

最有效的方法是画出积分区域的图像，观察其边界：

  1. 确定当前的积分次序，比如：

    ∫ab∫g1(x)g2(x)f(x,y) dy dx\int_a^b \int_{g_1(x)}^{g_2(x)} f(x,y) \, \mathrm{d} y \, \mathrm{d} x∫ab​∫g1​(x)g2​(x)​f(x,y)dydx

    表示对 yyy 先积分，其上界和下界是关于 xxx 的函数 g1(x)g_1(x)g1​(x) 和 g2(x)g_2(x)g2​(x)。

  2. 画出该区域，找出是否可以用另一种积分次序描述（如以 xxx 作为内积分变量）。

3. 确定新积分次序​

如果原积分次序是 ∫x∫y\int_x \int_y∫x​∫y​ ，交换次序后变成 ∫y∫x\int_y \int_x∫y​∫x​ ，则需要：

  • 找出在 yyy 方向上的变化范围（最小值到最大值）。
  • 对于每个固定的 yyy ，找到对应的 xxx 变化范围。

比如，考虑积分：

∫01∫x1f(x,y) dy dx\int_0^1 \int_x^1 f(x,y) \, \mathrm{d} y \, \mathrm{d} x∫01​∫x1​f(x,y)dydx
  • 原积分次序：对 yyy 先积分，它从 xxx 到 111 变化；xxx 从 0 到 1 变化。
  • 交换次序：
    • yyy 的范围是从 0 到 1。
    • 在固定的 yyy 下，xxx 变为从 0 到 yyy。
    • 新积分表达式： ∫01∫0yf(x,y) dx dy\int_0^1 \int_0^y f(x,y) \, \mathrm{d} x \, \mathrm{d} y∫01​∫0y​f(x,y)dxdy

4. 练习不同类型的积分区域​

常见的积分区域有：

  1. 矩形区域（最简单，直接交换次序）。
  2. 三角形区域（两条边界函数不同，需要换次序）。
  3. 由曲线或函数定义的区域（如抛物线、圆等）。
  4. 非标准区域（如椭圆、极坐标等情况）。

多做练习，掌握从图形到积分次序变换的能力。

5. 极坐标变换（适用于特殊情况）​

如果积分区域涉及圆形边界，考虑将二重积分换成极坐标：
