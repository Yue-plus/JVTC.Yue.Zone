# Kotlin

:::info
Kotlin 是一门现代但已成熟的编程语言，旨在让开发人员更幸福快乐。
它简洁、安全、可与 Java 及其他语言互操作，并提供了多种方式在多个平台间复用代码，以实现高效编程。

[官方网站](https://kotlinlang.org/docs/getting-started.html)
| [中文文档](https://book.kotlincn.net/text/getting-started.html)
:::
