# 积分表

## (一) 含有 $ax+b$ 的积分

01. $\int \frac{dx}{ax+b} = \frac{1}{a} \ln |ax+b| + C$

02. $\int (ax+b)^{\mu} dx = \frac{1}{a(\mu+1)} (ax+b)^{\mu+1} + C \quad (\mu \neq -1)$

03. $\int \frac{x}{ax+b} dx = \frac{1}{a^2} \left(ax + b - b \ln |ax+b| \right) + C$

04. $\int \frac{x^2}{ax+b} dx = \frac{1}{a^3} \left[ \frac{1}{2} (ax+b)^2 - 2b (ax+b) + b^2 \ln |ax+b| \right] + C$

05. $\int \frac{dx}{x(ax+b)} = -\frac{1}{b} \ln \left| \frac{ax+b}{x} \right| + C$

06. $\int \frac{dx}{x^2(ax+b)} = -\frac{1}{bx} + \frac{a}{b^2} \ln \left| \frac{ax+b}{x} \right| + C$

07. $\int \frac{x}{(ax+b)^2} dx = \frac{1}{a^2} \left( \ln |ax+b| + \frac{b}{ax+b} \right) + C$

08. $\int \frac{x^2}{(ax+b)^2} dx = \frac{1}{a^3} \left(ax + b - 2b \ln |ax+b| - \frac{b^2}{ax+b} \right) + C$

09. $\int \frac{dx}{x(ax+b)^2} = \frac{1}{b(ax+b)} - \frac{1}{b^2} \ln \left| \frac{ax+b}{x} \right| + C$
