## 对数运算法则

当 $a > 0, a \not = 1, M > 0, N > 0$ 时：

01. $\log_a a = 1$
02. $a^b = N \lrArr b = \log_a N$
03. $a^{log_a N} = N$
04. $a^x = e^{\ln a^x} = e^{x \ln a}$
05. $\log_a (MN) = \log_a M + \log_a N$
06. $\log_a \frac{M}{N} = \log_a M - \log_a N$
07. $\log_a M^x = x \log_a M$
08. $\log_a \frac{1}{M} = -\log_a M$
