## 关于班徽

![](/img/班徽.svg)

“R”代表“rally”团结 ，也可以代表软的拼音首字大写，代表着我们班犹如小船在大海中乘风破浪，同舟共济，扬帆起航。
宇航员代表着我们乘着学习 Java 的小船驶向星辰大海，“长风破浪会有时，直挂云帆济沧海”。
