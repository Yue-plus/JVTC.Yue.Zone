    2. 如果落点与起点不构成日字型，则不移动棋子；

    3. 如果落点处有自己方棋子，则不移动棋子；

    4. 如果在落点方向的邻近交叉点有棋子（绊马腿），则不移动棋子；

    5. 如果不属于1-4条，且落点处无棋子，则移动棋子；

    6. 如果不属于1-4条，且落点处为对方棋子(非老将)，则移动棋子并除去对方棋子；
