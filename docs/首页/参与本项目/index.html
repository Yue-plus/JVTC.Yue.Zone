参与本项目

本网站使用 Docusaurus 构建。

本项目使用 GPL-v3.0 协议在 GitHub 开源，请遵守开源协议。

邮件投稿​

可以直接将编写好的文档发送到 Yue_plus@foxmail.com， 由其他项目贡献者整理后上传到本网站。

邮件投稿的文档同样使用 GPL-v3.0 协议开源。

为方便筛选邮件，建议使用以下标题格式：

【悦域九职】[(docs/<path>)/blog] 备注

为方便其他项目贡献者整理编辑，建议使用以下文件格式：

  • 文本：.txt, .md, .docx, .html, .pdf, ...
  • 图片：.jpg, .png, .webp, ...
  • 多媒体文件请提供原文件附件
